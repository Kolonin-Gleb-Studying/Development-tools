Docker Compose — это средство, разработанное для помощи
в определении и совместном использовании многоконтейнерных приложений. 
С помощью средства Compose можно создать файл YAML.

Данный файл будет содержать все необходимые
переменные окружения и т.д., что сильно упростить построение контейнера.
